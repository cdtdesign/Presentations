Adipoli jQuery Image Hover Plugin

created by Joby Joseph in 2012



presented by Christina Thorpe

What does it do?

Adipoli jQuery Image Hover is a plugin that offers a simple option to provide a collection of unique and stylish hover effects to otherwise static images.


I personally like the plugin for its simplicity of installation and for the enhancements it can provide to images that may otherwise appear a bit boring.

What does the Adipoli plugin offer?


The plugin offers 15 unique hover effects, listed here:


• popout

• sliceDown

• sliceDownLeft

• sliceUp

• sliceUpLeft


• sliceUpRandom

• sliceUpDown

• sliceUpDownLeft

• fold

• foldLeft


• boxRandom

• boxRain

• boxRainReverse

• boxRainGrow

• boxRainGrowReverse

Here, the author demonstrates some of the hover effects: Adipoli Demo

Everybody LOVES Options!


So, available in the Adipoli plugin are a multitude of options that provide a wide range of customization.
Some of which include:

  • hoverEffect : Image style on mouse over
  • imageOpacity : opacity of image considered when start effect is transparent or overlay
  • animSpeed : Animation speed for the effect

Thus, MORE Options!!


  • Overlay color
  • Text Color
  • overlayText : Default HTML to be displayed on overlay
  • slices : Number of slices for slice animations
  • boxCols : Number of boxes in a row for box animations
  • boxRows : Number of rows for box animations
  • popOutMargin : Margin of Image popout
  • popOutShadow : Shadow length of popout image.


NOTE: Shadow works for those browsers which support text-shadow css

How is it to install?


Adipoli jQuery Image Hover Plugin is fairly simple to install…

First, enter the HTML code for the images that you want to place the hover effects on, making sure to ID each image, similar to my example below:


CODE VIEW


<div>
  <img class="img-style" id="image1" src="img/tcImage1.jpg"/>
  <img class="img-style" id="image2" src="img/tcImage2.jpg"/>
  <img class="img-style" id="image3" src="img/tcImage3.jpg"/>
<!-- and so on... -->
</div>

Next, place your CSS and JS links into your head tag:


CODE VIEW


<head>
  <link href="css/adipoli.css" rel="stylesheet" type="text/css"/>
  <script src="js/jquery-1.7.1.js" type="text/javascript"></script>
  <script src="js/jquery.adipoli.min.js" type="text/javascript"></script>
</head>

Create custom JS file. Place just before the end body tag. This will ensure all critical jQuery and JS has loaded prior to your custom options are in place.


CODE VIEW


<body>
  <!— your code —>

  <script type="text/javascript" src="js/yourJS.js"></script>
</body>

The custom JS document is where you will call on the Adipoli plugin and bind it to the images where the hover effects are desired.

Next, you must call the plugin’s function to the select images, as shown from my example below:


CODE VIEW


$(function() {

  $('#image1').adipoli();

Additionally, within this function you’ll want to insert the desired options you want applied to your image hovers. Place those directly inside the ( ) as shown in my example below:


CODE VIEW


$(function() {
	
  $('#image1').adipoli({
    'startEffect' : 'overlay',
    'hoverEffect' : 'foldLeft'
  });
  $('#image2').adipoli({
    'startEffect' : 'overlay',
    'hoverEffect' : 'foldLeft'
  });
  $('#image3').adipoli({
    'startEffect' : 'overlay',
    'hoverEffect' : 'foldLeft'
  });

My Example

You may achieve further customization
by tweaking the plugin’s master JS file (jquery.adipoli.js),
where you can edit the following rules:


CODE VIEW


(function($) {
  $.fn.adipoli = function(options) {
  // Create some defaults, extending them with any options that were provided
  //hovereffect: normal,sliceDown,sliceDownLeft,sliceUp,sliceUpLeft
  var settings = $.extend( {
    'startEffect'   : 'transparent',
    'hoverEffect'   : 'normal',
    'imageOpacity'  : 0.8125,
    'animSpeed'     : 500,
    'fillColor'     : '#3a2417',
    'textColor'     : '#fff',
    'overlayText'   : '',
    'slices'        : 10,
    'boxCols'       : 5,
    'boxRows'       : 3,
    'popOutMargin'  : 10,
    'popOutShadow'  : 10
  }, options);
